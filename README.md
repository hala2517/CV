## 🛠 Build

### Overleaf (권장)
1. [Overleaf](https://www.overleaf.com)에서 새 프로젝트 생성
2. `main.tex` 업로드
3. Compiler를 **XeLaTeX** 또는 **pdfLaTeX** 로 설정
4. Recompile

### Local
```bash
pdflatex main.tex
# 또는
xelatex main.tex
```

> 본 템플릿은 [altacv](https://github.com/liantze/AltaCV) 클래스를 사용합니다.
